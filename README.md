# to-do

このTo-Doアプリは、タスク管理を通して計画力と自己管理能力を向上させることを目的として開発されました。

このリポジトリには、自分で作成したTo-DoアプリのソースコードとREADMEを管理しています。

## Features
- Add tasks
- Mark tasks as completed
- Delete tasks

## Technologies Used
- HTML
- CSS
- JavaScript
- Python (Flask)

## How to Run
1. Clone the repository
2. Install required packages
3. Run app.py
4. Open browser and access localhost
